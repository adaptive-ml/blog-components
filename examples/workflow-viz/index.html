WorkflowViz Component

Interactive workflow visualization with three-tier detail levels

1. Interactive Workflow with Autoplay

The default configuration creates an interactive workflow that automatically progresses through steps. Users can also click nodes to navigate directly.

The user provides a natural language query.
"Plan a weekend trip in Copenhagen."
 <|im_start|> user Plan a weekend trip in Copenhagen <|im_end|>
The model generates internal reasoning to analyze the problem and plan its approach.
To plan a weekend trip, I need weather information and activity recommendations. I'll use get_weather to check conditions, then web_search for activities.
<|im_start|> assistant <think> To plan a weekend trip, I need weather information and activity recommendations. I'll use get_weather to check conditions, then web_search for activities. </think> <|im_end|>
The agent issues a tool call with function name and structured arguments.
{
  "name": "get_weather",
  "arguments": {
    "location": "Copenhagen, Denmark",
    "unit": "celsius"
  }
}
 <|im_start|> assistant <tool_call> {"name": "get_weather", "arguments": {"location": "Copenhagen, Denmark", "unit": "celsius"}} </tool_call> <|im_end|> <|im_start|> system <tool_response> {"temperature": 20, "condition": "Partly Cloudy", "humidity": 65, "wind_speed": 12} </tool_response> <|im_end|>
The agent calls another tool to gather activity recommendations.
{
  "name": "web_search",
  "arguments": {
    "query": "weekend activities Copenhagen"
  }
}
 <|im_start|> assistant <tool_call> {"name": "web_search", "arguments": {"query": "weekend activities Copenhagen"}} </tool_call> <|im_end|> <|im_start|> system <tool_response> {"results": [{"title": "Copenhagen Weekend Guide", "snippet": "Visit Tivoli Gardens, stroll through Nyhavn..."}]} </tool_response> <|im_end|>
The agent provides the final response to the user based on gathered information.
"Copenhagen will be 20°C and partly cloudy this weekend. I recommend visiting Tivoli Gardens, strolling through the colorful Nyhavn waterfront, or exploring Rosenborg Castle."
<|im_start|> assistant Copenhagen will be 20°C and partly cloudy this weekend. Component API

The WorkflowViz component supports the following attributes:

Attribute Type Default Description
interactive boolean true Enable autoplay and click navigation
step number 0 Initial step index (0-based)
expanded boolean false Show tier3 column by default
